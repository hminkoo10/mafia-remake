# Discord 마피아 봇

`/마피아시작`을 누르면 최대 1분 동안 참가 버튼을 띄우고, 버튼을 누른 유저에게 `마피아 참가자` 역할을 부여해 마피아, 의사, 경찰, 조커, 시민을 랜덤 배정하는 Discord 마피아 봇입니다. 봇이 밤 행동, 낮 토론, 투표, 승리 판정을 진행합니다.

낮 토론은 기본 1분 동안 먼저 진행됩니다. 1분이 지나면 10초 동안 `1분 연장` 투표가 열리고, 생존자 과반수가 누르면 1분 연장됩니다. 과반수가 모이지 않으면 바로 투표로 넘어갑니다.

## 기능

- `/마피아시작`은 누구나 사용 가능하며 최대 1분 참가자 모집 후 게임 시작
- `/마피아설정`으로 직업 수, 특수룰 수, 특수룰 활성화, 사망 직업 공개, 채팅 슬로우모드를 저장
- 참가자 모집 임베드는 비공개형 역할 구성, 최소 시작 인원, 실시간 참가 현황을 표시하고 `게임알림` 역할을 멘션
- 저장된 설정대로 `/마피아시작`이 진행되며, 기본 채팅 슬로우모드는 3초
- 주최자는 `시작` 또는 `취소` 버튼 사용 가능하며, 최소 인원 미달 시 자동 취소
- `/마피아중지`로 진행 중인 게임 강제 종료
- `/마피아상태`로 현재 생존자/사망자 확인
- `/마피아능력`으로 역할 배정 후 언제든 자기 역할과 전체 능력 설명 확인
- `/역할안내`로 게임 진행 여부와 상관없이 역할 안내를 공지용 임베드로 전송
- `마피아중지`와 `마피아상태`는 `관리자` 역할을 가진 유저만 사용 가능
- 참가 버튼을 누른 유저에게만 `마피아 참가자` 역할 자동 부여
- 역할은 DM으로 비공개 발송
- 시작 전/시작 직후 공개 역할 구성은 마피아, 의사, 경찰, 시민 계열과 특수 인원 수만 표시
- 봇이 보내는 안내/결과 메시지는 임베드로 표시
- 마피아/경찰/의사 전용 비공개 채널 자동 생성
- 특수 역할 비공개 채널에 과반 규칙 안내 자동 공지
- 조커는 투표로 처형되면 즉시 단독 승리
- 마피아/의사/경찰 밤 행동은 DM 선택 메뉴로 처리
- 밤 행동 대상자가 모두 선택하면 밤 타이머를 기다리지 않고 바로 아침 진행
- 밤 시간이 10초 남으면 게임 채널에 안내
- 경찰 조사의 성공 여부는 낮에 공개하고, 실제 조사 결과는 경찰 비공개 채널로 전달
- 낮 토론은 1분 뒤 생존자 10초 과반 투표로 1분 연장 또는 투표 진행
- 낮 투표는 시작 채널의 선택 메뉴로 처리하며 스킵 가능
- 생존자가 모두 투표하면 투표 타이머를 기다리지 않고 바로 정산
- 낮 투표 뒤 지목자는 20초 최후변론을 하고, 이후 찬반투표 과반 결과에 따름
- 최후변론 중에는 지목자만 말할 수 있고 슬로우모드는 해제되며, 찬반투표 뒤 설정된 슬로우모드가 다시 적용됨
- 투표 결과에는 누가 누구를 찍었는지 없이 대상별 익명 득표 수 표시
- 특수룰: 사립탐정, 스파이, 도굴꾼, 대부, 조커, 정치인, 테러리스트 지원
- 정치인은 최후변론/찬반투표까지 진행되며, 투표권은 2표로 계산되고 처형 찬성이 우세하면 직업 공개 후 처형되지 않고 밤으로 넘어감
- 테러리스트는 밤마다 지목 대상을 고르고, 마피아 팀에게 사망하거나 투표 처형될 때 조건에 맞으면 대상도 함께 사망
- 스파이는 밤마다 직업을 확인하고, 선택한 대상이 마피아라면 접선한 뒤 같은 밤 첩보를 한 번 더 사용
- 대부는 세 번째 밤에 마피아 팀과 자동 접선한 뒤 확정 처치 능력을 사용
- 밤과 투표 시간에는 `마피아 참가자` 역할의 게임 채널 채팅이 비활성화됨
- 낮 토론 시간에는 `마피아 참가자` 역할만 채팅 가능
- 사망자는 설정에 따라 직업 공개 또는 비공개 처리되며, `마피아 참가자` 역할을 제거하고 채팅/반응 및 해당되는 역할별 비공개 채널 권한도 비활성화
- 게임 종료 시 게임 채널 권한을 복구하고 모든 참가자의 `마피아 참가자` 역할 제거 및 역할별 비공개 채널 삭제

## 실행 방법

1. Discord Developer Portal에서 봇을 만들고 `Server Members Intent`를 켭니다.
2. 봇 초대 URL에 `bot`과 `applications.commands` scope를 포함합니다.
   단계별 채팅 제한과 역할별 비공개 채널을 쓰려면 봇에게 `채널 관리` 권한을 주세요.
   봇이 임베드를 보낼 수 있도록 `링크 첨부` 권한도 허용하세요.
   참가자 역할 자동 부여/제거를 쓰려면 봇에게 `역할 관리` 권한을 주고, 봇 역할을 `마피아 참가자` 역할보다 위에 두세요.
3. `.env.example`을 `.env`로 복사한 뒤 `DISCORD_TOKEN`에 봇 토큰을 넣습니다.
4. 서버에 `마피아 참가자`, `관리자` 역할을 만듭니다. 모집 알림 멘션을 쓰려면 `게임알림` 역할도 만듭니다.
5. 의존성을 설치하고 봇을 실행합니다.

```bash
pip install -r requirements.txt
python bot.py
```

## 명령어

```text
/마피아시작
/마피아설정 mafia:2 doctor:1 police:1 slowmode:3
/마피아설정 citizen_special:1 mafia_special:1 neutral_special:1
/마피아능력
/역할안내
/마피아상태
/마피아중지
```

투표 스킵 선택지 때문에 참가자는 최대 24명까지 지원합니다. 기본 역할 이름, 직업 수, 밤/토론/투표 시간은 `config.json`에서 바꿀 수 있습니다.

마피아/의사/경찰이 여러 명이면 살아있는 같은 역할 인원 기준 과반 초과가 같은 대상을 골라야 밤 행동이 행사됩니다. 동률이거나 과반에 못 미치면 해당 행동은 실행되지 않습니다.
